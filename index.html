Research

Cancer3D: 3D hybrid mechanistic model of tumor growth

A coupled continuous–discrete framework for simulating solid tumour dynamics in three dimensions. The continuous phase models nutrient diffusion and average cell density via PDEs; the discrete phase tracks individual cell fate decisions (proliferation, migration, apoptosis) as a stochastic agent-based system. Validated against in vitro growth curves across receptor density conditions.

Active C++ Python Agent-based modelling PDEs → GitHub

Machine learning for tumour growth prediction

Supervised learning models trained on tumour kinetics data to predict growth trajectories and therapeutic response. Combines classical regression approaches with deep learning to complement mechanistic models with data-driven inference.

Python PyTorch Scikit-learn Predictive modelling
